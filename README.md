# D&D Silent Master Bot

Telegram-бот для отправки скрытых событий игрокам в D&D.

## Особенности

- ✅ Полноценные листы персонажей D&D 5e
- ✅ Интеграция с D&D 5e API для рас и классов
- ✅ Автоматическое применение расовых и классовых особенностей
- ✅ Расчет характеристик и модификаторов
- 🔲 Система комнат для Мастеров (в разработке)
- 🔲 Скрытые сообщения игрокам (в разработке)

## Установка

1. Установи зависимости:
```bash
pip install -r requirements.txt
```

2. Создай `.env` файл на основе `.env.example`:
```bash
copy .env.example .env
```

3. Получи токен у @BotFather в Telegram и добавь его в `.env`:
```
BOT_TOKEN=твой_токен_здесь
```

4. Запусти бота:
```bash
python bot.py
```

## Текущий функционал (Этапы 1-2)

- ✅ Создание персонажа с полными характеристиками D&D 5e
